- Node v13

- MariaDB v10.2
- MongoDB

- cURL
- OpenSSL
- `pip` for Python 3
- `zip` / `unzip`

The following dev/admin tools are included, too:

- Git
- htop
- MailDev
- Ngrok
- tmux
- Vim
