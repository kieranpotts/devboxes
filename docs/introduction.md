# Introduction

Vagrant boxes are pre-packaged development environments. The boxes are compatible with different virtualization providers such as VirtualBox, VMWare, Hyper-V and Docker.

Most Vagrant boxes are just naked operating systems, intended to form a "bare metal" baseline for provisioning with system and application software. However, the Vagrant boxes built from the source code in this code repository are a fully-provisioned web servers for various server-side software stacks.
