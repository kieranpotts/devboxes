# Dev Boxes

This is a monorepo that encapsulates the source code for several different Vagrant-provisioned local development environments for various software stacks.

The development environments are packaged as Vagrant boxes and distributed publicly via [Vagrant Cloud](https://app.vagrantup.com/kieranpotts/).
